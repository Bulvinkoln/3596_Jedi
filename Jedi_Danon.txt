Напишу я сюда инструкцию, дабы не запамятовал как выпихивать да впихивать файлы
1. Для начала зайдем в GitHub, найдем файл нам нужный, вилканем его к себе 
2. Находим ссылку, заходим в GIT, открываем терминал, через команду git clone https/// типа загружает папку и пишет что она пуста
3. Открываем File -> Open Folder -> ищем папку которую клонировали
4. **ОБЯЗАТЕЛЬНО** создаем свою ветку
5. В этой ветке создаем файл
6  Git add .
7. Добавляем коммиты, чет пишем важное там 
8. заливаем к себе в GitHub git push -u origin name_of_your_branch
